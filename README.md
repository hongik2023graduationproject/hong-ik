# hong-ik: 한글 프로그래밍 언어 프로젝트

## 소개

**hong-ik**은 한글을 기반으로 설계된 자체 프로그래밍 언어 프로젝트입니다. C++로 작성된 트리워킹 인터프리터와 바이트코드 VM을 모두 지원합니다.

## 설치 및 실행 방법

### 필요한 환경
- C++26 이상 지원 컴파일러
- [CMake](https://cmake.org/) 3.x 이상

### 빌드
```bash
cmake -B cmake-build-debug
cmake --build cmake-build-debug
```

### 테스트
```bash
ctest --test-dir cmake-build-debug
```

### 프로그램 실행
```bash
# 파일 실행
./hong-ik filename.hik

# REPL 모드
./hong-ik

# 바이트코드 VM 모드
./hong-ik --vm filename.hik
```

## 언어 문법

### 변수 선언
```
정수 x = 10
실수 pi = 3.14
문자 이름 = "홍길동"
논리 참 = true
배열 목록 = [1, 2, 3]
사전 정보 = {"이름": "홍길동", "나이": "25"}
```

### Null 타입
```
정수 x = 없음
x == 없음    // true
```

### 문자열 보간
```
문자 이름 = "홍길동"
문자 인사 = "안녕 {이름}님"    // "안녕 홍길동님"
```

### 조건문
```
만약 x > 5 라면:
    출력("크다")
아니면:
    출력("작다")
```

### 반복문 (while)
```
반복 i < 10 동안:
    i += 1
```

### 범위 반복문 (for-range)
```
반복 정수 i = 0 부터 10 까지:
    출력(i)
```

### ForEach 반복문
```
각각 정수 원소 [1, 2, 3] 에서:
    출력(원소)
```

### 함수 선언 및 호출
```
함수 더하기(정수 a, 정수 b) -> 정수:
    리턴 a + b

더하기(3, 7)    // 10
```

### 기본 매개변수
```
함수 인사(문자 이름, 문자 접미사 = "님") -> 문자:
    리턴 이름 + 접미사

인사("홍길동")        // "홍길동님"
인사("홍길동", "씨")  // "홍길동씨"
```

### 고차 함수 (Lambda/Higher-order)
```
함수 두배(정수 x) -> 정수:
    리턴 x * 2

매핑([1, 2, 3], 두배)         // [2, 4, 6]
걸러내기([1, -2, 3], 양수)    // [1, 3]
줄이기([1, 2, 3], 합, 0)      // 6
```

### 비교/매칭 (Match/Switch)
```
비교 x:
    경우 1:
        출력("일")
    경우 2:
        출력("이")
    기본:
        출력("기타")
```

### 튜플
```
(1, 2, 3)[0]    // 1
```

### 클래스 / OOP
```
클래스 동물:
    문자 이름
    정수 나이
    생성(문자 이름, 정수 나이):
        자기.이름 = 이름
        자기.나이 = 나이
    함수 소개() -> 문자:
        리턴 자기.이름

동물 강아지 = 동물("뽀삐", 3)
강아지.이름      // "뽀삐"
강아지.소개()    // "뽀삐"
```

### 상속
```
클래스 강아지 < 동물:
    함수 소리() -> 문자:
        리턴 "멍멍"

강아지 뽀삐 = 강아지("뽀삐", 3)
뽀삐.소개()    // 부모 메서드 호출 가능
뽀삐.소리()    // "멍멍"
```

### 예외 처리
```
시도:
    정수 x = 10 / 0
실패 오류:
    출력(오류)
```

### 모듈 가져오기
```
가져오기 "파일명.hik"
```

### 음수 인덱싱
```
배열 목록 = [10, 20, 30]
목록[-1]    // 30
목록[-2]    // 20
```

## 바이트코드 VM

`--vm` 플래그를 사용하면 트리워킹 인터프리터 대신 바이트코드 VM으로 실행합니다.

```bash
./hong-ik --vm filename.hik
```

VM은 소스코드를 바이트코드로 컴파일한 후 스택 기반 가상 머신에서 실행합니다. 기본 연산, 변수, 함수, 클래스, 예외 처리 등 대부분의 기능을 지원합니다.
