# Changelog

이 프로젝트의 주요 변경 사항을 기록합니다. [Keep a Changelog](https://keepachangelog.com/ko/1.0.0/) 형식을 따릅니다.

## [0.7.0] - 2026-03-20

### 추가
- 거듭제곱 연산자(`**`) 지원
- 증감 연산자(`++`, `--`) 지원
- VM 단락 평가(short-circuit evaluation) 구현

### 수정
- UTF-8 문자열 처리 개선
- 문자열 이스케이프 시퀀스 수정
- null 역참조, 배열 범위 초과(OOB) 접근 시 안전 처리
- 상속 시 필드 초기화 버그 수정
- 파일 경로 보안 검증 강화

### 테스트
- 테스트 10건 추가

## [0.6.0] - 2026-03-15

### 추가
- `계속`(continue)문 구현
- 람다(Lambda) 지원
- 메서드 체이닝 지원
- VM 동등성(parity) 확대 -- 트리워킹과 VM 모드 간 기능 일치 개선

### 수정
- 보안/안정성 코드 감사 32건 전체 수정

## [0.5.0] - 2026-03-15

### 추가
- 클래스 상속 (`클래스 자식 < 부모`)
- 고차 함수: `매핑`, `걸러내기`, `줄이기`
- 기본 매개변수 지원 (`함수 인사(문자 이름, 문자 접미사 = "님")`)
- 범위 반복문 (`반복 정수 i = 0 부터 10 까지`)
- ForEach 반복문 (`각각 정수 원소 [1,2,3] 에서`)
- 비교/매칭문 (`비교`/`경우`/`기본`)
- 예외 처리 (`시도`/`실패`)
- 모듈 가져오기 (`가져오기 "파일명.hik"`)
- 에러 복구 기능

## [0.4.0] - 2026-03-15

### 추가
- 바이트코드 VM 구현 (`--vm` 플래그)
  - 스택 기반 가상 머신
  - 바이트코드 컴파일러
  - VM 상수 폴딩 최적화
  - VM 클로저 및 REPL 세션 지원
- 수학 내장 함수 11개: `사인`, `코사인`, `탄젠트`, `로그`, `자연로그`, `거듭제곱`, `파이`, `자연수e`, `반올림`, `올림`, `내림`
- JSON 내장 함수 2개: `JSON_파싱`, `JSON_직렬화`
- 문자열 내장 함수 5개: `시작확인`, `끝확인`, `반복`, `채우기`, `부분문자`
- 음수 인덱싱 지원 (`목록[-1]`)
- 튜플 타입 지원
- 문자열 보간 (`"안녕 {이름}님"`)

### 수정
- 트리워킹 파일 모드 수정
- 에러 메시지 한국어화 및 개선 (줄 번호 포함)

## [0.3.0] - 2026-03-15

### 추가
- 언어 대규모 확장 및 문법 개선
  - 사전(Dictionary) 타입
  - 클래스/OOP (`클래스`, `생성`, `자기`)
  - Null 타입 (`없음`)
- 내장 함수 대폭 확대 (28개 -> 46개)
  - 수학: `절대값`, `제곱근`, `최대`, `최소`, `난수`
  - 문자열: `분리`, `대문자`, `소문자`, `치환`, `자르기`
  - 배열: `정렬`, `뒤집기`, `찾기`, `조각`
  - 타입 변환: `정수변환`, `실수변환`, `문자변환`
  - 기타: `타입`, `키목록`, `포함`, `설정`, `삭제`, `파일읽기`, `파일쓰기`

## [0.2.0] - 2026-02-28

### 변경
- raw pointer에서 smart pointer(shared_ptr)로 전환
- 크리티컬 메모리 관련 버그 수정

## [0.1.0] - 2025-07-01 ~ 2025-07-22

### 추가
- 한글 키워드 기반 프로그래밍 언어 기본 구현
  - 변수 선언: `정수`, `실수`, `문자`, `논리`, `배열`
  - 조건문: `만약`/`라면`/`아니면`
  - 반복문: `반복`/`동안`
  - 함수 선언 및 호출: `함수`, `리턴`
  - Boolean 연산자: `&&`, `||`
- 기본 내장 함수: `출력`, `길이`, `추가`, `입력`
- REPL 모드 및 파일 모드 지원
- UTF-8 한글 인코딩 처리
- Google Test 기반 테스트 인프라 구축
  - Lexer, Parser, UTF-8 Converter 테스트

### 수정
- Visual Studio UTF-8 컴파일 옵션 추가
- 구문 미완성 시 nullptr 참조 방지 (NoTokenException)
- 문자열 닫는 따옴표 누락 시 에러 처리

## [0.0.1] - 2024-12-17 ~ 2025-06-30

### 추가
- 프로젝트 초기 생성
- Lexer: 사칙연산, 정수, 식별자, 예약어 토큰화
- Parser: 사칙연산 AST 생성, prefix 연산자
- Evaluator: 정수 사칙연산 실행
- REPL 초안 구현
- `.clang-format` 코드 스타일 설정
