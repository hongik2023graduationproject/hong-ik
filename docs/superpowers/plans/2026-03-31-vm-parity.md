- [ ] **Step 4: Modify `compileFunction` to emit default value prologue**

Instead of using separate "default chunks" that require re-entering `run()` (which would corrupt frame state), we compile default values as **conditional bytecode at the start of the function body**. For each parameter with a default, we emit: check if arg slot is a sentinel → if so, evaluate default expression and store.

However, since the VM pushes defaults *before* creating the frame (simpler approach), we store pre-evaluated constant defaults in `CompiledFunction`. For non-constant defaults, we compile them inline.

**Approach: store default `Object` values for constant expressions (covers 95% of cases).**

In `vm/chunk.h`, the `defaultValues` field stores pre-evaluated defaults:

```cpp
int defaultCount = 0;
std::vector<std::shared_ptr<Object>> defaultValues; // size == arity, nullptr for required params
```

In `vm/compiler.cpp`, modify `compileFunction` (around line 807) — after declaring locals for params:

```cpp
// After: for (auto& param : stmt->parameters) { declareLocal(param->name); }
// Pre-evaluate constant default values:
funcState.function->defaultValues.resize(stmt->parameters.size());
int defaultCount = 0;
for (size_t i = 0; i < stmt->defaultValues.size(); i++) {
    if (stmt->defaultValues[i]) {
        defaultCount++;
        // Extract constant value from literal expressions
        if (auto* intLit = dynamic_cast<IntegerLiteral*>(stmt->defaultValues[i].get())) {
            funcState.function->defaultValues[i] = make_shared<Integer>(intLit->value);
        } else if (auto* floatLit = dynamic_cast<FloatLiteral*>(stmt->defaultValues[i].get())) {
            funcState.function->defaultValues[i] = make_shared<Float>(floatLit->value);
        } else if (auto* strLit = dynamic_cast<StringLiteral*>(stmt->defaultValues[i].get())) {
            funcState.function->defaultValues[i] = make_shared<String>(strLit->value);
        } else if (auto* boolLit = dynamic_cast<BooleanLiteral*>(stmt->defaultValues[i].get())) {
            funcState.function->defaultValues[i] = make_shared<Boolean>(boolLit->value);
        } else if (dynamic_cast<NullLiteral*>(stmt->defaultValues[i].get())) {
            funcState.function->defaultValues[i] = make_shared<Null>();
        } else {
            // Non-constant default: evaluate at compile time is not safe
            // For now, throw an error. Future: emit inline bytecode prologue.
            throw RuntimeException("VM에서는 상수 기본값만 지원합니다.", 0);
        }
    }
}
funcState.function->defaultCount = defaultCount;
```

// Helper (add as private method in vm.h):
void fillDefaults(CompiledFunction* fn, int argCount) {
    int requiredArgs = fn->arity - fn->defaultCount;
    if (argCount < requiredArgs || argCount > fn->arity) {
        throw RuntimeException("함수가 필요한 인자 개수와 입력된 인자 개수가 다릅니다.", currentLine());
    }
    for (int i = argCount; i < fn->arity; i++) {
        if (fn->defaultValues[i]) {
            push(fn->defaultValues[i]);
        } else {
            push(make_shared<Null>());
        }
    }
}
```

// Index assignment: 변수[인덱스] = 값
if (current_token->type == TokenType::IDENTIFIER && next_token != nullptr
    && next_token->type == TokenType::LBRACKET) {
    // Lookahead to find matching ] followed by =
    auto pos = current_read_position;
    int bracketDepth = 0;
    bool isIndexAssign = false;
    for (auto p = pos + 1; p < static_cast<long long>(tokens.size()); p++) {
        if (tokens[p]->type == TokenType::LBRACKET) bracketDepth++;
        else if (tokens[p]->type == TokenType::RBRACKET) {
            if (bracketDepth == 0) {
                if (p + 1 < static_cast<long long>(tokens.size()) && tokens[p + 1]->type == TokenType::ASSIGN) {
                    isIndexAssign = true;
                }
                break;
            }
            bracketDepth--;
        }
    }
    if (isIndexAssign) {
        return parseIndexAssignmentStatement();
    }
}
```

- [ ] **Step 5: Implement `parseIndexAssignmentStatement`**

In `parser/parser.cpp`, add:

```cpp
shared_ptr<IndexAssignmentStatement> Parser::parseIndexAssignmentStatement() {
    auto stmt = make_shared<IndexAssignmentStatement>();

    // Parse collection (identifier)
    stmt->collection = make_shared<IdentifierExpression>();
    dynamic_cast<IdentifierExpression*>(stmt->collection.get())->name = current_token->text;
    dynamic_cast<IdentifierExpression*>(stmt->collection.get())->token = current_token;
    skipToken(TokenType::IDENTIFIER);

    // Parse [index]
    skipToken(TokenType::LBRACKET);
    stmt->index = parseExpression(Precedence::LOWEST);
    setNextToken();
    skipToken(TokenType::RBRACKET);

    // Parse = value
    skipToken(TokenType::ASSIGN);
    stmt->value = parseExpression(Precedence::LOWEST);
    setNextToken();
    skipToken(TokenType::NEW_LINE);

    return stmt;
}
```

case OpCode::OP_IMPORT: {
    uint16_t nameIdx = readUint16();
    auto* filename = dynamic_cast<String*>(frame.function->constants[nameIdx].get());
    if (!filename) throw RuntimeException("잘못된 파일명입니다.", currentLine());

    // Canonicalize path for dedup
    std::string canonicalPath;
    try {
        canonicalPath = std::filesystem::weakly_canonical(filename->value).string();
    } catch (...) {
        canonicalPath = filename->value;
    }

    // Skip if already imported
    if (importedFiles.count(canonicalPath)) break;
    importedFiles.insert(canonicalPath);

    // Read file
    std::ifstream file(filename->value);
    if (!file.is_open()) {
        throw RuntimeException("파일을 열 수 없습니다: " + filename->value, currentLine());
    }
    std::stringstream buffer;
    buffer << file.rdbuf();
    std::string source = buffer.str();

    // Lexer -> Parser -> Compiler pipeline
    Lexer importLexer;
    std::vector<std::shared_ptr<Token>> allTokens;
    std::istringstream stream(source);
    std::string line;
    int indent = 0;

    while (std::getline(stream, line)) {
        line += "\n";
        auto utf8 = Utf8Converter::Convert(line);
        auto tokens = importLexer.Tokenize(utf8);
        allTokens.insert(allTokens.end(), tokens.begin(), tokens.end());
        for (auto& t : tokens) {
            if (t->type == TokenType::START_BLOCK) indent++;
            if (t->type == TokenType::END_BLOCK) indent--;
        }
    }
    for (int i = 0; i < indent; i++) {
        allTokens.push_back(std::make_shared<Token>(Token{TokenType::END_BLOCK, "", 0}));
    }

    Parser importParser;
    auto program = importParser.Parsing(allTokens);

    Compiler importCompiler;
    auto importedFn = importCompiler.Compile(program);

    // Execute imported code in current VM context
    CallFrame importFrame;
    importFrame.function = importedFn.get();
    importFrame.ip = 0;
    importFrame.slotOffset = stack.size();
    importFrame.hasCallee = false;

    // Store ownership of the compiled function (keep alive)
    importedModules.push_back(importedFn);

    frames.push_back(importFrame);
    break;
}

bool Compiler::containsYield(BlockStatement* block) {
    if (!block) return false;
    for (auto& stmt : block->statements) {
        if (dynamic_cast<YieldStatement*>(stmt.get())) return true;
        if (auto* whileStmt = dynamic_cast<WhileStatement*>(stmt.get())) {
            if (containsYield(whileStmt->body.get())) return true;
        }
        if (auto* ifStmt = dynamic_cast<IfStatement*>(stmt.get())) {
            if (containsYield(ifStmt->consequence.get())) return true;
            if (containsYield(ifStmt->then.get())) return true;
        }
        if (auto* forEachStmt = dynamic_cast<ForEachStatement*>(stmt.get())) {
            if (containsYield(forEachStmt->body.get())) return true;
        }
        if (auto* forRangeStmt = dynamic_cast<ForRangeStatement*>(stmt.get())) {
            if (containsYield(forRangeStmt->body.get())) return true;
        }
    }
    return false;
}

void Compiler::compileYield(YieldStatement* stmt) {
    compileExpression(stmt->expression.get());
    chunk().emitOp(OpCode::OP_YIELD, 0);
}

- [ ] **Step 11: Modify `OP_RETURN` for generator functions**

In `vm/vm.cpp`, in the `OP_RETURN` handler (around line 435), before pushing the result to the caller.

**Important:** Return an `Array`, NOT an `IteratorState`. The caller's `각각...에서` loop will call `OP_ITER_INIT` on the return value, which already knows how to create an `IteratorState` from an `Array`. Returning `IteratorState` directly would cause double-wrapping and break iteration.

```cpp
case OpCode::OP_RETURN: {
    auto result = pop();
    size_t slotOffset = currentFrame().slotOffset;
    bool hasCallee = currentFrame().hasCallee;
    bool wasGenerator = currentFrame().isGenerator;
    auto yieldValues = std::move(currentFrame().yieldBuffer);
    frames.pop_back();

    while (stack.size() > slotOffset) {
        stack.pop_back();
    }

    if (frames.empty()) {
        if (wasGenerator) {
            auto arr = make_shared<Array>();
            arr->elements = std::move(yieldValues);
            return arr; // Return Array, not IteratorState
        }
        return result;
    }

    if (hasCallee && !stack.empty() && stack.size() > currentFrame().slotOffset) {
        stack.pop_back();
    }

    if (wasGenerator) {
        auto arr = make_shared<Array>();
        arr->elements = std::move(yieldValues);
        push(arr); // Push Array — OP_ITER_INIT will wrap it
    } else {
        push(result);
    }
    break;
}
```
